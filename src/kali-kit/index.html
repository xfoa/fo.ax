foax's Kali Kit

Networking in MS-DOS is complicated and scary. To make things a bit easier, I've put together a collection of tools, drivers, scripts, and guides to easily set up networking and Kali IPX to TCP/IP tunneling in DOS. Get it on GitHub!

Included in the kit:

  • Guides on how different kinds of network drivers work in DOS, and how to set them up
  • Packet drivers for a wide range of network adapters
  • ODI drivers for popular network adapters
  • IPX drivers
  • TCP/IP tools
  • The latest versions of Kali for DOS and Windows
  • A guide to host networking for 86Box on Linux using VDE, plus scripts to help set it up
  • Documentation available in both HTML and DOS compatible TXT formats!

Either use it as pre-built ISO, burn it to CD, or build your own custom version with the included scripts!
